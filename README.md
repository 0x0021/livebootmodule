# LiveBoot Magisk Module

A Magisk module that enables unix-style boot animation for Android devices - Magisk module of Chainfire's [LiveBoot](https://github.com/Chainfire/liveboot) app without any app or UI

Preview: https://www.youtube.com/watch?v=N0tqzVWxpJk

Tested on Mi 10T Pro with PixelExperence Plus 13. Should work all AOSP based custom ROMs.
